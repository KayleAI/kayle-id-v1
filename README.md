## Getting Started

Kayle ID enables you to verify the identity or perform age verification of your users.

## Contributing

## Security

If you believe you have found a security vulnerability in Kayle ID, responsibly disclose it by emailing [security@kayle.id](mailto:security@kayle.id). Do not open a public issue or pull request.

## Data Processing

Kayle ID processes end-user identity data to perform document checks.

We minimise what we store, encrypt all verification results, and never use this data for any purpose other than verification. You can read more about our data processing in our [privacy policy](https://kayle.id/privacy).
